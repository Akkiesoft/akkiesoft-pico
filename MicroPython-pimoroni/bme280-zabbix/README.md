# Sending BME280 data to Zabbix

This script reads the data that reading BME280 and send to Zabbix by Zabbix Sender format.

## Things necessary

* Raspberry Pi Pico ( https://www.raspberrypi.org/products/raspberry-pi-pico/ )
    * or other Pico compatible boards
* Pimoroni Pico Wireless Pack ( https://shop.pimoroni.com/products/pico-wireless-pack )
* MicroPython build by Pimoroni ( https://github.com/pimoroni/pimoroni-pico )
